# Basic Linux Commands

## Check Current Directory

```bash
pwd
```

Displays the current working directory.

---

## List Files

```bash
ls
```

Lists files and directories.

---

## List Hidden Files

```bash
ls -a
```

Shows all files including hidden files.

---

## Change Directory

```bash
cd directory_name
```

Moves to another directory.

---

## Create Directory

```bash
mkdir folder_name
```

Creates a new directory.

---

## Create File

```bash
touch filename.txt
```

Creates an empty file.

---

## Remove File

```bash
rm filename.txt
```

Deletes a file.

---

## Remove Directory

```bash
rmdir folder_name
```

Removes an empty directory.

---

## Clear Terminal

```bash
clear
```

Clears the terminal screen.
