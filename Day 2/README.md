# 📅 Day 2 – Linux File System

Welcome to **Day 2** of my Linux learning journey!

Today, I explored the **Linux File System**, its directory hierarchy, and essential navigation commands. Understanding the Linux file system is a fundamental step toward becoming proficient in Linux administration and DevOps.

## 📚 Topics Covered

* Linux File System Overview
* Root (`/`) Directory
* Linux Directory Hierarchy
* Important System Directories

  * `/home`
  * `/root`
  * `/bin`
  * `/sbin`
  * `/etc`
  * `/var`
  * `/usr`
  * `/boot`
  * `/dev`
  * `/proc`
  * `/sys`
  * `/tmp`
  * `/media`
  * `/mnt`
  * `/opt`
* File Navigation Commands
* Directory Exploration
* Basic File System Practice

## 🛠️ Commands Practiced

```bash
pwd
ls
ls -l
ls -la
cd
cd ..
cd ~
tree
```

## 🎯 Learning Outcome

By the end of Day 2, I was able to:

* Understand the Linux directory structure.
* Navigate efficiently through the file system.
* Identify the purpose of important system directories.
* Use common navigation commands with confidence.
* Explore system information stored in virtual file systems like `/proc` and `/sys`.
